# ⚡ Быстрый старт LawArk

## 🎯 Что создано

Полноценный Telegram бот для юридических консультаций с AI:

### ✅ Реализованные функции:
- 💬 Юридические консультации с Deepseek AI
- 📄 Анализ документов (PDF, DOCX, изображения)
- 🔍 OCR распознавание текста (Tesseract)
- 💳 Система подписок с тарифами
- 📊 Статистика использования
- 🔗 Кросс-промо с HealthArk
- 🗄️ SQLite база данных
- 📱 Современный UI с клавиатурами

## 🚀 Запуск за 5 минут

### 1. Установите зависимости:
```bash
pip install aiogram==3.4.1 aiosqlite==0.19.0 python-dotenv==1.0.0 Pillow==10.2.0 pytesseract==0.3.10 PyPDF2==3.0.1 python-docx==1.1.0 httpx==0.26.0
```

### 2. Установите Tesseract OCR:

**Windows:**
- Скачайте: https://github.com/UB-Mannheim/tesseract/wiki
- Установите в `C:\Program Files\Tesseract-OCR`

**Linux:**
```bash
sudo apt-get install tesseract-ocr tesseract-ocr-rus -y
```

**macOS:**
```bash
brew install tesseract tesseract-lang
```

### 3. Настройте токены в `.env`:
```env
BOT_TOKEN=ваш_токен_от_BotFather
DEEPSEEK_API_KEY=ваш_ключ_от_Deepseek
DATABASE_PATH=lawark.db
MAX_FILE_SIZE_MB=20
```

### 4. Запустите бота:
```bash
python main.py
```

## 🎮 Как пользоваться

1. **Запустите бота** командой `/start`
2. **Выберите действие**:
   - 💬 Задать вопрос - текстовые консультации
   - 📄 Анализ документа - загрузите файл
   - 💳 Подписка - выберите тариф
   - 📊 Статистика - ваши данные

3. **Получите AI-ответ** с анализом и рекомендациями

## 🔧 Возможности

### Юридические консультации:
- Анализ договоров и документов
- Разъяснение законов РФ
- Рекомендации по правовым вопросам
- Структурированные ответы с ссылками на статьи

### Анализ документов:
- **Изображения**: JPG, PNG → OCR распознавание
- **PDF**: извлечение текста
- **DOCX**: анализ документов Word
- **Размер**: до 20MB

### AI интеграции:
- **Deepseek API**: для консультаций
- **Tesseract OCR**: для распознавания текста
- **Telegram Bot API**: для взаимодействия

## 🛠 Технические детали

- **Язык**: Python 3.9+
- **Фреймворк**: aiogram 3.x
- **База данных**: SQLite (aiosqlite)
- **AI**: Deepseek API
- **OCR**: Tesseract
- **Архитектура**: Модульная (handlers/services/database)
